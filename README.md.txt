# 🖼️ Susovan Image Editor

A desktop image editor built using **C++ + Qt + OpenCV**, featuring real-time image processing with an intuitive UI.

---

## 🚀 Features

- 🎚️ Brightness & Contrast adjustment  
- 🎨 RGB color control (cinematic scaling)  
- 🧠 Blur & Sharpen filters  
- ✂️ Manual + Mouse crop (rectangle overlay)  
- 🔄 Rotate image (custom degrees)  
- 🔁 Flip (Horizontal / Vertical)  
- ⚡ Edge detection  
- 🔄 Undo / Redo support  
- 🖱️ Drag & Drop image loading  
- 📂 Click-to-browse modern UI

## ⚙️ Build Instructions

### Requirements:
- Qt (6.x recommended)
- OpenCV installed
- CMake

### Steps:

```bash
git clone https://github.com/YOUR_USERNAME/Susovan_Image_Editor.git
cd Susovan_Image_Editor
mkdir build
cd build
cmake ..
cmake --build .
